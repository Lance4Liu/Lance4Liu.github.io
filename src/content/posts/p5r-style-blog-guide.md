本文将记录如何从零开始，搭建一个具有 P5R 风格的个人博客。从设计思路、技术选型到具体实现，带你一步步实现梦想中的站点。

## 1. 设计灵感

《女神异闻录 5 皇家版》（P5R）以其独特的视觉风格给我留下了深刻印象。强烈的红黑配色、铆钉与撕纸元素、动感排版与 UI 设计，充满了反叛与个性的气息。

> 我们追求的不是被认可，而是颠覆常规的自由。  
> —— JOKER

因此，我希望我的博客也能传达这种态度：简洁而不失个性，专注内容本身，但在细节处体现风格。

## 2. 技术栈

本博客使用以下技术构建：

- Astro — 现代静态站点生成器，构建速度极快
- Tailwind CSS — 原子化 CSS 框架，快速实现设计稿
- TypeScript — 类型安全的 JavaScript，减少运行时错误
- MDX — 让 Markdown 拥有组件能力，内容表达更丰富

## 3. 核心视觉实现

P5R 风格的核心在于「撕裂感」和「强对比」。在 CSS 中，我使用 `clip-path` 实现不规则切角，用多层背景与噪点纹理强化画面冲击力。

- 背景：噪点纹理 + 径向渐变模拟油漆飞溅
- 卡片：不规则切角 + 白色描边 + 红色阴影
- 字体：Bebas Neue（展示）+ Oswald（标题）+ Noto Sans SC（正文）
- 动画：尽量使用 transform / opacity，确保性能稳定

## 4. 性能优化

视觉效果丰富不代表性能差。通过合理使用图片尺寸、避免重排动画、减少无意义的客户端脚本，页面在 Lighthouse 中依旧保持了较高分数。
